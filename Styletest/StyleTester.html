History

HTML was developed as a universal markup language that can be used to display content(s) on websites.
Due to the widespread usage of the internet websites can be viewed across a wide array of electronic devices of various shapes and sizes.
Hence there is a need for consistent display of webpages, therefore the W3 organisation at MIT (which established various standards
across the World Wide Web) established HTML5 that is standardised and they have entirely eliminated the concept of versions for HTML to stremline with the development of the world wide web.

General Tags

    DOCTYPE tag
    This tag is compulsary in an HTML file as it indicates the version of HTML document. Currently it indicates version 5.
    HTML tag
    It is essential as the entire content of the file should be enclosed within this tag.
    head tag
    This tag contains user defined information about the html document. This information is enclosed inside another tag called "title".
    The page related style SELECTORS should also be defined here.
    Body tag
    As the name indicates this tag contains the body i.e. the main structural content of the file.

Content tags

    P tag
    This tag contains the paragraph information of an html file.
    Heading tag
    All the headings related to a html file must be defined here, it has the following tag names h1,h2,h3,h4,h5 and h6 in the
    decreasing order of size.
    List tag
    Lists are extremely useful in html as they are highly effective in indicating point-wise information.

Semantec tags

  • em - used to make characters italic
  • code - used to indicate program syntaxes
  • abbr - indicates abbreviations
  • strong - used to bold the corresponding characters
